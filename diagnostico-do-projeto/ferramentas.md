# HTML: Quais elementos estão sendo usados?
Os elememtos utilizados foram:
h1: para ajustar a fonte, de modo que fique compatível com o site e se mostre mais atraente/chamativo
h2: para oferecer o uma melhor compatibilidade e alinhamento
<p>: para definir e construir os textos
<ul>: para pôr alguns elementos mais importantes de forma ordenada e listada para que fique organizada
<strong>: para dar destsque em algumas partes do texto
<em>: para ofertar partes importantes em destaque
img: para construir um bom ajustamento entre imagens e tesxtos
links: para trazer e levar a outras partes do site
form: para produzir o formulário de inscrição ao entrar na página
div: para ajudar a dar um maior agrupamento na página entre os elementos
<button>: para que uma pessoa aperte ao concluir o formulário

# CSS: Quais propriedades estão sendo usadas?
background-color: para poder mudar a cor de fundo de um elemento
font-family: para mudar a fonte do elemento do site
display: para poder definir como mostrar os elementos
flex: utilizado para flexionar/fixar o formulário do site
gap: utilizado para modificar o tamanho da brecha das palavras
width: foi usado para definir o comprimento do formulário em sua largura
justify-content: foi usado para definir uma margem no formulário
align-items: para alinhar as letras no centro
font-weight: para definir o tamanho da fonte que foi utilizada
justify-self: usado para justificar a posição do site
font-style: para mudar a fonte dos elementos que foram usados ao CSS
text-decoration: para exibir alguma decoração no texo (porém, não utilizei nenhuma decoração)
position: para deixar a posiçaõ da página relativa ao site
transform: para transformar a translação dos elementos do site
left: para calcular o lado esquerdo do site
margin: para implementar uma margem nas bordas dos elementos
padding: foi usado para conferir o preenchimento dos elementos do site
box-sizing: usado para definir o tamanho da borda do formulário
color: para adicionar uma cor na letra no título do site
